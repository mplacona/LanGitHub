# Langithub - v 0.1
---
Search for public repositories on github by language 

## Demo
[Langithub om Placona.co.uk](http://langithub.placona.co.uk/)

## Dependencies
* [nodejs](http://nodejs.org)
* [npm](https://npmjs.org/)

## Quick start
- Clone or download this repo.
- Run `npm install`
- Run `node app.js`
- Access your app in `http://localhost:3000`
